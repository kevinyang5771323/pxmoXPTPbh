# 智能图书管理系统 java866

## 项目概述


## 技术栈

### 后端技术

- Spring Boot 3 + MyBatis-Plus：高性能Java后端框架
- JWT + Spring Security：安全认证体系
- Redis：缓存和会话管理
- 协同过滤算法：实现智能图书推荐
- MySQL：数据持久化存储

### 前端技术

- Vue 3 + Pinia：现代化前端框架和状态管理
- Element Plus + Vant：美观的UI组件库
- ECharts：数据可视化展示
- WangEditor：富文本编辑功能
- Axios：HTTP请求处理

## 功能模块

### 用户端功能

1. **用户画像**：分析读者阅读偏好
2. **智能推荐**：基于协同过滤算法的个性化图书推荐
3. **留言板**：读者交流互动平台
4. **图书查找**：快速定位所需书籍
5. **借阅管理**：借阅历史记录和当前借阅状态
6. **收藏管理**：个人图书收藏夹
7. **订阅管理**：新书订阅功能
8. **系统通知**：及时获取图书馆动态
9. **个人中心**：个人信息管理

### 管理端功能

1. **仪表盘**：数据概览和统计分析
2. **用户管理**：读者信息管理
3. **图书管理**：图书CRUD操作
4. **借阅管理**：借阅记录审批
5. **订阅管理**：读者订阅处理
6. **通知管理**：系统消息推送
7. **书架管理**：物理书架维护
8. **书籍分类**：图书分类体系
9. **用户留言**：读者反馈处理
10. **黑名单管理**：违规用户管理

## 项目特色

- **智能推荐系统**：基于用户行为和偏好的协同过滤算法，推荐更精准
- **完善的权限控制**：RBAC模型实现精细化的权限管理
- **响应式设计**：适配PC和移动端
- **丰富的可视化**：ECharts实现数据直观展示
- **安全可靠**：JWT认证+加密传输保障数据安全
